# 高级 React 设计模式

## 1. 组合组件模式 (Compound Components Pattern)
组合组件是一种模式，其中组件协同工作，从而共享隐式状态。

jsx
const Tabs = ({ children }) => {
  const [activeTab, setActiveTab] = useState(0);
  return (
    <TabContext.Provider value={{ activeTab, setActiveTab }}>
      {children}
    </TabContext.Provider>
  );
};


## 2. 渲染属性模式 (Render Props Pattern)
渲染属性是一个函数属性，组件使用它来了解要渲染的内容。

jsx
<DataProvider render={data => (
  <h1>你好 {data.name}</h1>
)} />
